函数式编程之纯函数

纯函数的定义是，对于相同的输入，永远会得到相同的输出，而且没有任何可观察的副作用，也不依赖外部环境的状态，及是不依赖，不修改其作用域之外的变量的函数

纯函数实例：bug守恒定律
